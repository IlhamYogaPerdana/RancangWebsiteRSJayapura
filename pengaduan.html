Pertanyaan yang Sering Diajukan

Berapa lama waktu respon untuk pengaduan yang diajukan?
Tim kami berkomitmen untuk merespon setiap pengaduan dalam waktu maksimal 1x24 jam (hari kerja). Untuk keluhan darurat, respon akan diberikan dalam waktu maksimal 4 jam.
Apakah identitas pelapor akan dirahasiakan?
Ya, kami menjamin kerahasiaan identitas pelapor. Informasi pribadi Anda hanya akan digunakan untuk keperluan penyelesaian pengaduan dan tidak akan disebarluaskan.
Bagaimana cara melacak status pengaduan saya?
Setelah pengaduan dikirim, Anda akan menerima nomor tiket via email. Gunakan nomor tersebut untuk melacak status pengaduan melalui sistem kami atau hubungi customer service.
Apakah ada biaya untuk mengajukan pengaduan?
Tidak ada biaya untuk mengajukan pengaduan. Layanan ini gratis sebagai bagian dari komitmen kami terhadap kepuasan pasien dan peningkatan kualitas layanan.
Jenis file apa saja yang bisa dilampirkan?
Anda dapat melampirkan file berformat JPG, PNG, PDF, DOC, atau DOCX dengan ukuran maksimal 10MB per file dan maksimal 5 file per pengaduan.
